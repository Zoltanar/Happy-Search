Filtering

This section can be used to filter the results in the main tab:

  • Each filter can be toggled on or off and you can save the combination of filters to be used later (These groupings of filters are called Custom Filters).
  • Each filter can also be marked as 'Fixed' which means that it will not change when a Custom Filter is selected.
    This is useful if there are filters which you want to be on at all times (such as hide blacklisted items).
  • On the filters which have multiple possible options (such as Userlist), you can select a single option by double clicking it,
    so you don't have to deselect each other option.
  • You can also switch between 'Tags AND Traits' mode and 'Tags OR Traits' mode, depending on your preference.
  • Information on tags and traits filtering is available in each section respectively.
