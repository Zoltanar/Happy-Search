Searching and Filtering

This section can be used to find new titles and filter the current results by means other than those in the Tag Filtering section.

  • To the left of the this button you can list titles already in your local database (Go Button) or get new ones from VNDB (Update Button).
  • Below and to the left is the Connection Status panel and other methods of listing titles.
  • To the right is the section where you filter the current list (along with the tag and trait filtering sections above).

In the explanations below, all titles listed are in local database, not in VNDB, unless stated otherwise.

Listing

List by Name:

By clicking the Go button, list all titles that match specified name or original name (alias to be added later).
By clicking the Update button, get all titles that match specified name, original name or alias, from VNDB.

List by Year:

By clicking the Go button, list all titles released in specified year.
By clicking the Update button, get all titles released in specified year, from VNDB.

List by Producer:

By clicking the Go button, lists all titles by specified producer entered by user.

By clicking the Update button, get all titles by specified producer, from VNDB.

List by Group:

Lists titles that are in selected user-defined group.

Filtering

Tag/Trait Filter Indicators

The small squares indicate whether the tag and/or trait filters, respectively, are active.
Red means they are active and gray means they are not. Click an indicator to clear the respective filter.

The button in between indicates whether the filters are used in 'AND' mode or 'OR' mode.
'AND' mode will list titles that BOTH have all tags in the active tag filter AND characters with all traits in the active trait filter.
'OR' mode will list titles that EITHER have all tags in the active tag filter OR characters with all traits in the active trait filter.

URT

Select whether to show user-related titles, hide them, show ONLY User-related titles, or only show titles that user hasn't finished or dropped.

Unreleased

Select whether to show unreleased titles, hide them, show ONLY unreleased titles or show all titles except those with no release date set (a partial release date is still accepted).

Blacklisted

Select whether to show blacklisted titles, hide them, or show ONLY blacklisted titles.
