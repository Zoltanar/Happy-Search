Searching and Filtering

This section can be used to find new titles and filter the current results by means other than those in the Tag Filtering section.

  • The section to the left of the Help button lets you search for VNs by name or release year.
  • Below and to the left is the section where you choose the list of VNs to be displayed.
  • To the right is the section where you filter the current list (along with the tag filtering section above).

In the explanations below, all titles listed are in local database, not in VNDB, unless stated otherwise.

Listing

List by Name:

By pressing enter on the textbox, list all titles with name or original name matching what the user entered (alias to be added later).
By clicking the Update button, get all titles with name, original name or alias matching what the user entered, from VNDB.

List by Year:

By pressing enter on the textbox, list all titles released in specified year.
By clicking the Update button, get all titles released in specified year, from VNDB.

All Titles

List all titles.

Favorite Producers

List all titles by favorite producers.

Wishlist Status

List all titles with the selected wishlist status.

Userlist Status

List all titles with the selected userlist status.

List by Producer:

Lists titles by producer entered by user.

List by Group:

Lists titles that are in group with the name entered by user.

Update

Gets new titles by producer from VNDB that aren't already in local database.
If a producer name is entered that isn't in local database, it will ask user to search VNDB for it.
This can be used to list a producer's titles without adding that producer to the user's favorites.

Filtering

Tag/Trait Filter Indicators

The small squares indicate whether the tag and/or trait filters, respectively, are active.
Red means they are active and gray means they are not. Click an indicator to clear the respective filter.

The button in between indicates whether the filters are used in 'AND' mode or 'OR' mode.
'AND' mode will list titles that BOTH have all tags in the active tag filter AND characters with all traits in the active trait filter.
'OR' mode will list titles that EITHER have all tags in the active tag filter OR characters with all traits in the active trait filter.

URT

Select whether to show user-related titles, hide them, show ONLY User-related titles, or only show titles that user hasn't finished or dropped.

Unreleased

Select whether to show unreleased titles, hide them, or show ONLY unreleased titles.

Blacklisted

Select whether to show blacklisted titles, hide them, or show ONLY blacklisted titles.

Other

View Picker

Select whether to view results in tile view (default) or detailed view.
