Getting Started

To get started, you will want  to log into VNDB by clicking the 'Log In' button, you can log in two different ways:

  • Entering your User ID which is displayed in your user page in VNDB.org (https://vndb.org/u####)
  • Entering your User ID and VNDB.org login credentials

The application may be used by different users, identified by their User ID, favorite producers are saved by User ID, however, other settings are saved per application.

Using login credentials will give you the ability to change Userlist status, Wishlist priority and/or Vote on Visual Novels, it will also allow you to get your userlist/wishlist/votelist if you have them set as hidden.

If you aren't using login credentials and attempt to get your User-related titles but no results are found, your lists may be hidden, either login or change the setting and you might have to wait up to 25 hours.


Please note that features that connect with vndb.org can be throttled which means that Happy Search must wait some minutes before continuing to connect.
You can continue using the program's features which don't connect with vndb.org while waiting, such as listing and filtering.
The status of the connection to vndb.org is displayed in the Status Box, described below.

Status Box

This is the box just above the results list, at the left, it tells you the status of the connection to vndb.org:

  • Ready, this means the connection is ready to be used and you may use the features that connect with vndb.org
  • Busy, this means the connection is being used at the moment, you cannot do anything during this time.
  • Throttled, this means the connection is not being used but it has received too many commands from Happy Search
    and must wait before continuing, you may use features that don't connect with vndb.org during this time.
    The program will continue executing the method that caused the throttling automatically after enough time has passed.
  • Error or Closed, this means connection was closed or failed to open, you can try again by attempting to Log In with the button above.

User-Related Titles (URT)

These are titles in your userlist, wishlist and votelist.

Buttons and Settings

Log In

Described above, you can check the 'Remember' checkbox to save your credentials to be used later (password is encrypted).

Update List

This will get all your user-related titles from VNDB.org, it will take a while if you have a lot of titles and this is the first time, the update is additive so only new titles are added, therefore consecutive updates should not take long.

Other Functions

This is a dropdown list of functions that shouldn't be used regularly, for solving errors or changes in past releases.

  • Get All VN Stats
    Get data about each title's stats, it only needs to be used if you updated from version older than 1.2
  • Get All Character Data
    Get data about each title's characters (traits) this can take a while, it only needs to be used if you updated from version older than 1.3
  • Get Missing Covers
    Get cover images for titles in the local database, useful if you modify or replace the database externally but do not replace the cover images folder to match.
  • Update Title Data (All)
    Updates tags traits and stats of all titles, regardless of their release date or date of last update.
    Only do this if you don't mind the potentially long time that it could take or if your local database is small.

Update Title Data

This will update the tags, traits and stats of all titles that haven't been updated in at least 7 days, it may take a while if there are a lot of titles.

The number of days between updates is based on the release date of the titles:

Titles released in last 6 months = 7 days.

Titles released 6 months - 1 year ago = 14 days.

Titles released 1 year - 2 years ago = 28 days.

Titles released 2-10 years ago = 56 days.

Titles released over 10 years ago aren't updated.

Titles released by favorite producers are also updated every 7 days.

Show NSFW Images

Off by default, when turned on, NSFW images will be displayed.

Auto-update URT

Off by default, when turned on, user's URT will be updated every 2 days upon application launch

10 Year Limit

Off by default, when turned on, it will limit VNs fetched from VNDB to those released in the last 10 years, this can reduce throttling. Getting VNs by favorite producer or searching by name will bypass this.

Close All VN Windows

If you open a lot of VN windows (by clicking the tiles in the list) you can close all of them at once with this button.
