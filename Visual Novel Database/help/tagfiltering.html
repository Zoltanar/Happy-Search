Tag Filtering

This section can be used to filter the Visual Novel list by tags:

  • Left side contains the ten most common tags for the current list of Visual Novels, you can check the boxes to add that tag to the filters.
  • You can also add tags by name by typing in the box at the top, it is case-insensitive and if you wish to use the auto-suggestions you must type in order (Typing 'Female' will suggest 'Female Protagonist' but not 'Strong Female Protagonist')
  • Right side will show current tag filters, you can uncheck the boxes to remove the filters.
  • You can save a selection of filters as a custom filter in order to quickly filter by the saved tags instead of having to add them one by one each time.

Buttons and Boxes

Tag Type Boxes

These are the three small boxes at the top, they let you choose which types of tags should be shown.

Tag Search Box

You can type the name of a tag here and hit enter or click on a suggestion to add it as a filter.

Custom Filter Name Box

If you wish to save the list of active filters, type a name here and hit Save Filter or press enter.

Save Filter

Save the list of active filters as a custom filter.

Delete Filter

Delete the active custom filter.

Clear Filter

Clear the list of active filters (does not modify custom filters).

Update Filter

Get all VNs that match the list of active filters which were not already in local database, if '10 Year Limit' is enabled, only titles released in the last 10 years will be fetched.
