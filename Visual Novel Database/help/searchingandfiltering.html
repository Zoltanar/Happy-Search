Searching and Filtering

This section can be used to find new titles and filter the current results by means other than those in the Tag Filtering section.

  • The section to the left of the Help button lets you search for VNs.
  • Below and to the left is the section where you choose the list of VNs to be displayed.
  • To the right is the section where you filter the current list (along with the tag filtering section above).

In the explanations below, all titles listed are in local database, not in VNDB, unless stated otherwise.

Searching

Search by Name:

Get all titles with name, original name or alias matching what the user entered, this gets titles from VNDB, not just in local database.

Get Titles by Year:

Get all titles released in the year entered by user, this gets titles from VNDB if they aren't already in local database.

Listing

All Titles

List all titles.

Favorite Producers

List all titles by favorite producers.

Wishlist Status

List all titles with the selected wishlist status.

Userlist Status

List all titles with the selected userlist status.

List by Producer:

Lists titles by producer entered by user.

Update

Gets new titles by producer from VNDB that aren't already in local database.

Filtering

URT

Select whether to show user-related titles, hide them, show ONLY User-related titles, or only show titles that user hasn't finished or dropped.

Unreleased

Select whether to show unreleased titles, hide them, or show ONLY unreleased titles.

Blacklisted

Select whether to show blacklisted titles, hide them, or show ONLY blacklisted titles.

Other

View Picker

Select whether to view results in tile view (default) or detailed view.
