Favorite Producers

This section allows you to add producers to a list in order to quickly filter titles based on the producers and provide features such as:

  • Display titles developed by your favorite producers
  • Get your average vote and drop rate on a producer's titles
  • Display upcoming titles by your favorite producers

Take note that in the context of this application, a title's producer is the developer.


Buttons

Add Producers

This will bring up a window where you can search for producers by name and add the selected ones to your list.

Show VNs From Selected

This will display all titles developed by the selected producers.

Remove Selected

This will remove the selected producers from your list.

Get New Producer Titles

This will get all titles (by favorite producers) from VNDB that are not already in local database.

Update All Producer Titles

This will get all titles (by favorite producers) from VNDB including ones already in local database, it usually takes much longer than only getting new titles.

Load Unloaded

This is like Get New Producer Titles but only goes through unloaded producers.


Terms

Developer

A title may have multiple releases, each with a different developer, in order to automatically choose a developer, it is determined to be the developer of the first non-trial release.

Average Vote

This is the average score based on your votes on a producer's titles.
Only titles with votes are counted.

Drop Rate

This is the percentage of titles which you have dropped, only counting a producer's dropped and finished titles.
This can be a useful statistic if you drop titles because they are bad, but choose not to vote because you have not completed the game.
Drop Rate = (number of dropped titles) / (number of dropped titles + number of finished titles)
For example, you finished 3 titles and dropped 1, the drop rate is 25% (1/4).

Loaded

This is whether a producer has had it's titles fetched from VNDB since producer was added to your list.
Since it may take some time to load a producer's titles for the first time, it is not done automatically when a producer is added to your list.
