Listing and Searching

This section can be used to list or find new titles.
In the explanations below, all titles listed are in local database, not in VNDB, unless stated otherwise.

Listing

List by Name:

By clicking the Go button, list all titles that match specified name, original name or alias.
By clicking the Update button, get all titles that match specified name, original name or alias, from VNDB.

List by Year:

By clicking the Go button, list all titles released in specified year.
By clicking the Update button, get all titles released in specified year, from VNDB.

List by Producer:

By clicking the Go button, lists all titles by specified producer entered by user.

By clicking the Update button, get all titles by specified producer, from VNDB.

List by Group:

Lists titles that are in selected user-defined group.

All Titles

List all titles.

Filters

You can select a filter for the results, you can learn more about these in the filters tab.

Other

View Picker

Select whether to view results in tile view (default) or details view (a line by line display).

Multi Actions

Perform actions on all selected titles. Titles can be selected by Ctrl+clicking them and they are identified by a golden border.

  • Deselect All - Clears selection of titles.
  • Remove From DB - Remove titles from local database, also removes associated images.
  • Update tags/traits/stats - Updates tags/traits/stats for selected titles.
  • Update all data - Updates all data for selected titles.
